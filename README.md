# GSC Salzgitter e.V. im ADAC - Website Prototyp

Dies ist ein Prototyp für die Website des **GSC Salzgitter e.V. im ADAC**. 

Die Website wurde mit **Antigravity** entwickelt und befindet sich aktuell noch im Aufbau. Diese erste Version dient in erster Linie dazu, erste technische Deployment-Tests durchzuführen.
