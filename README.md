# GSC Salzgitter e.V. im ADAC - Website

Dies ist ein Minimal-Viable-Product für die Website des **GSC Salzgitter e.V. im ADAC**. 

Der Code für diese Website wurde mit **Antigravity** entwickelt. Diese erste Version stellt die nötigsten Informationen in einem übersichtlichen Format bereit. Weitere Inhalte folgen in nächster Zeit in Abhängigkeit von Kapazitäten und Prioritäten in Abstimmung mit dem Vorstand.
